## HuHinka's Blog Source

[博客]使用 [Hexo] 搭建并部署在 GitHub Pages 上。

主题选用 [hexo-theme-icalm]。

### 环境配置

```
npm install -g hexo-cli
npm install
```

### 编写/发布博客

```
// 创建新博客草稿
hexo n 'new blog title'
// 查看草稿
npm run draft
// 发布博客
hexo publish 'new blog title'
```

### 部署到 GitHub Pages

```
npm run deploy
```
